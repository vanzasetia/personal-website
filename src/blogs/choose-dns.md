Using the right public Domain Name System (DNS) servers is helpful because you can block nasty links. But I do not stick to one DNS provider. I keep changing my DNS providers whenever I want because I do not trust any of them with *all* my data.

Here are the DNS providers that I have been using since 2021:

- [NextDNS](https://nextdns.io/) allows us to use it for free without creating an account and to have a wide range of selections for blocklists. We can use a temporary email, like [Guerrilla Mail](https://www.guerrillamail.com/), to make a permanent account. I like this very much, and it is easy to set up.
- [Cloudflare DNS](https://developers.cloudflare.com/1.1.1.1/setup/) is the fastest DNS that I have ever used so far. The problem is that we can not see what Cloudflare is blocking, and we can not control what we want to block and allow.
- [AdGuard DNS](https://adguard-dns.io/en/public-dns.html) works well without any problems at all. But just like Cloudflare, we can not see what AdGuard blocks and allows.
- [Quad9](https://quad9.net/) has a unique way to secure its users. In rare cases, it becomes slow.

I do not trust the DNS from my ISP at all because it does not even try to limit how much data they collect. Not only that, but they also block DuckDuckGo. I would rather use one of them or [any random public DNS servers](https://adguard-dns.io/kb/general/dns-providers/).
