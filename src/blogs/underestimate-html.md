[HTML Hell](https://www.htmhell.dev/) exists to show people that HTML is hard to get right.

You need to understand the effect of your HTML markup. For example, the `<br>` element causes some screen readers to say, "break." It is annoying.

So do not think that you are an HTML expert just because you know the syntax and remember all the HTML elements.

Never underestimate HTML.
