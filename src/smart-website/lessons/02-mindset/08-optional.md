title: Almost everything is optional
---

CSS is optional. JavaScript can be forgotten. Content? You can have a blank page for your website.

How do people make things become required? Their goals decide the requirements. If you want to make a really beautiful personal website, your requirements are to use CSS and learn to make compelling visual design.

To decide your requirements, you need to have one or more goals. The more goals you have, the more requirements you need. The fewer goals you have, the fewer requirements you need.

Set a goal. Then, you will know what you need to do.
