title: Do we need to sort CSS properties?
---

You do not need to sort the CSS property in any way. Unless you work in a group or want to do that, sorting CSS properties is optional.

You can use [CSS Declaration Sorter](https://www.npmjs.com/package/css-declaration-sorter) or [csscomb](https://www.npmjs.com/package/csscomb). But, for simplicity's sake, do not use them.
